# 🌈 Bunq high-Fidelity Design Audit

This document serves as the visual "Source of Truth" for aligning the **Bunqsy Dashboard** with the authentic Bunq mobile application aesthetic.

## 1. Core Typography: The "Bunq Balance"
- **Primary Value**: Large, bold font weight.
- **Decimal Treatment**: Sub-pixel precision using **superscript decimals** (e.g., € 1,227.²⁴). Period separator for thousands, comma for decimals.
- **Currency Symbols**: Sourced from the specific local account (e.g., €, $, A$, ¥). Always positioned to the left of the value with a single space.

## 2. Iconography & Component Tiles
- **The "Tile" Pattern**: Icons are hosted inside large, rounded-square backgrounds with vibrant, saturated colors (Goldenrod, Sky Blue, Forest Green).
- **Icon Style**: Use high-quality 3D-effect emojis or simple white silhouettes centrally positioned within the tile.
- **Card Backgrounds**: Predominantly **Dark Mode Pro** (#0F172A style). High-contrast borders (1px solid rgba(255,255,255,0.1)).

## 3. Categorized Spending & Budgeting
- **Pie Charts**: Multi-color segments (Purple, Orange, Yellow, Red) with circular category icons floating on the perimeter, aligned to their respective segments.
- **Budgeting Limits**: Use a **Pink Dashed Reference Line** across bar charts with a clearly readable "LIMIT: €XXX" label pinned to the right.
- **Staggered Lists**: For spending distribution, use cards with varying background widths to create a "pulsing" or staggered visual rhythm.

## 4. Global & Multi-Currency Views
- **Currency Cards**: Use high-contrast background tiles (Deep Red for ES, Navy for US, Dark Green for AUS) to differentiate accounts at a glance.
- **The Global Theme**: For international features, use the Earth/Globe illustration as a backdrop or watermark to represent "Borderless Banking."

## 5. Security & Trust Architecture
- **Urgent Notifications**: "Stay Safe" alerts use a centralized card pattern with a distinct Bunq-rainbow logo and concise, bold copy.
- **Intervention Visuals**: 
    - **Fraud**: Padlock icons with 3D depth and high-luminance blue glows.
    - **Verify**: Magnifying glass rimmed in Purple over specific fields (like IBAN matching).
- **State Feedback**: Use literal illustrations (Lightning bolts for API/Live data, Paper planes for Sent transactions).

## 6. ESG & Impact Integration
- **Planting Trees**: Use the "Green Globe" icon (Earth with leaves) to represent the account's carbon footprint or reforestation impact.
- **Consistency**: All impact metrics should use the signature Bunq Green (#00C896).

## 7. Interactive Elements
- **Quick Action Buttons**: Large circular buttons with simple icons (+, ↑, ↓).
- **The "Sleep" Trigger**: footer-based toggle using the 🌙 emoji and a glowing loading bar for state transitions.
- **Voice Command Orb**: The central rainbow orb representing the "Bunqsy" guardian.
